# Bootstrap Assignment

## Overview
This assignment focuses on learning and implementing Bootstrap, a popular front-end framework for building responsive web applications.

## Objectives
- Understand Bootstrap's grid system
- Learn component usage (buttons, cards, forms, navigation)
- Create responsive layouts
- Style web pages efficiently
