
# CSS Position and Flex

## Overview
This directory contains assignments and training materials focused on CSS positioning and flexbox layout techniques.

## Topics Covered
- **CSS Position**: static, relative, absolute, fixed, sticky positioning
- **Flexbox**: flex containers, flex items, alignment, and layout patterns

## Getting Started
1. Review the assignment requirements
2. Create HTML and CSS files as needed
3. Test your layouts across different screen sizes
