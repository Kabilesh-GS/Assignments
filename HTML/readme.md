## Overview

Two standalone HTML pages:

- A personal landing page for Kabilesh with about and contact sections.
- An accessibility-focused contact form demo.

## Files Included

- `LandingPage.html` - Personal landing page with navigation, about section, social links, and contact form with senmatics and ARIA.
- `Accessibility.html` - Accessibility form example with semantic structure and ARIA-required inputs.

## Features

- Responsive meta viewport and basic semantic structure.
- Contact forms using fieldsets, legends, and labeled controls.
- Accessibility considerations: `aria-required` flags and descriptive alt text placeholders.
- Social link placeholders for quick external navigation.
