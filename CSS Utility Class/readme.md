# CSS Utility Class and Design Tokens

## Overview
This project demonstrates the use of CSS utility classes and Design Tokens for rapid styling and development.

## Contents
- Basic utility class and design token examples
- Common use cases
- Best practices

## Getting Started
1. Review the utility class definitions
2. Apply classes to your HTML elements
3. Customize as needed for your project

## Features
- Responsive design utilities
- Spacing and sizing helpers
- Color and typography utilities
- Layout utilities (flexbox)

## Usage
```html
<div class="flex justify-center items-center">
  <p class="text-lg font-bold">Example</p>
</div>
```
